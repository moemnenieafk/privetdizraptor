## [2.4.0] - Интеграция API и раздел "Нужные предметы"

### 🚀 Добавлено (Added)
- **Интеграция API (tarkov.dev):** Страница "Нужные предметы" (`/eft/keepitems`) теперь подтягивает реальные данные с барахолки. Реализован алгоритм автоматического присвоения редкости (Обычное, Редкое, Эпическое, Легендарное) на основе стоимости предмета.
- **Новые роуты:** Создана страница-заглушка `/eft/tracker`, уведомляющая о переносе радара в глобальный `TacticalSearch`.

### ♻️ Изменено (Changed)
- **Очистка роутинга:** Полностью удален устаревший маршрут `/eft/progression/itemtracker/page.tsx` для предотвращения конфликтов и ошибок сборки.

## [2.3.0] - Архитектура FSD и Единый Командный Центр

### 🚀 Добавлено (Added)
- **FSD Архитектура:** Кодовая база реструктурирована по методологии Feature-Sliced Design. Выделены слои `layout`, `ui`, `features` и `providers`.
- **Командная панель (Command Palette):** `TacticalSearch.tsx` преобразован в умный выпадающий список (Dropdown), объединяющий быструю навигацию по разделам сайта и глобальный поиск предметов EFT.
- **Динамический роутинг:** Внедрен `[gameId]/page.tsx` для автоматической обработки хабов всех игр на основе `GAMES_DATA`, статические папки заглушек удалены.

### ♻️ Изменено (Changed)
- **Удаление дублей:** Компонент `ItemSearch.tsx` удален, так как его функционал полностью поглощен глобальным `TacticalSearch`.
- **Глобализация Layout:** Компоненты `<Header />` и `<Footer />` вынесены в корневой `layout.tsx`, устранены дублирования на внутренних страницах.

## [2.2.0] - Динамический Хедер и Тактический поиск

### 🚀 Добавлено (Added)
- **Глобальный Хедер:** Внедрен умный двухстрочный компонент `Header.tsx`, который автоматически подхватывает тему и логотип текущей игры через `usePathname`.
- **Конфигурация навигации:** Добавлен словарь `src/data/headerConfig.ts` для управления ссылками, плейсхолдерами и валютой в зависимости от активного раздела (EFT, FRAGO, ABI и др.).
- **Тактический поиск:** Создан клиентский компонент `TacticalSearch.tsx` с поддержкой операционных систем (CMD+K / CTRL+K) и неоновым фокусом.
- **Телеметрия:** Внедрен `PlayerTelemetry.tsx` для отображения уровня, прогресса и баланса пользователя с динамической валютой.
- **Структура проекта:** Создан файл `PROJECT_STRUCTURE.md` для удобного отслеживания архитектуры и путей в проекте.

### ♻️ Изменено (Changed)
- Захардкоженный `layout.tsx` для EFT теперь использует новый универсальный `<Header />`.

## [2.1.0] - Глобальный тактический хаб и оптимизация

### 🚀 Добавлено (Added)
- **YouTube API:** Внедрен `Server Action` для динамической подгрузки свежих видео с каналов Fullkamen и BSG с кэшированием (1 час).
- **LiteYouTube:** Создан фасадный компонент для ленивой загрузки тяжелых `iframe`, что радикально улучшило производительность главной страницы.
- **Навигация:** Внедрен компонент `Breadcrumbs` (Хлебные крошки) для удобной навигации внутри многоуровневого раздела `/eft`.
- **Страницы игр:** Созданы заглушки для всех новых проектов (`/abi`, `/gzw`, `/frago` и т.д.) с поддержкой видео-фонов, каскадных анимаций текста и динамическим акцентом темы.
- **Анимации перехода:** В проект добавлен `src/app/template.tsx` для плавной анимации (`fade-in`) при переходе между страницами (роутинге).
- **Вложенность:** Добавлен раздел `/eft/progression` для логической разгрузки главной страницы EFT.

### ♻️ Изменено (Changed)
- **Тактическая сетка EFT:** Весь раздел `/eft` переведен на строгую сетку (колонки 160px, отступ 28px).
- **HubCard:** Карточки хаба переписаны с использованием `aspect-ratio` и поддерживают два адаптивных размера (`square` и `rectangle`). Внедрены CSS-маски для перекрашивания иконок.
- **Достижения:** Страница `/eft/achievements` полностью переверстана под новую сетку `1100px` с использованием обновленных дизайн-токенов.
- **Карусель:** В `Carousel.tsx` добавлена поддержка горизонтальной прокрутки колесиком мыши (с защитой от инерции тачпадов Apple) и адаптивные размеры для мобильных экранов (`80vw`).
- **Прелоадер:** На главную страницу возвращен экран тактической предзагрузки ("СИНХРОНИЗАЦИЯ").
- **Фон:** Глобальный паттерн (точки) сделан более чистым и разреженным (`53px` шаг, `1px` толщина), убрана затемняющая виньетка для предотвращения муара.

### 🐛 Исправлено (Fixed)
- **Twitch API:** Серверный обработчик `route.ts` теперь кэширует OAuth токен в памяти, предотвращая блокировку сервера со стороны Twitch (Rate Limits).
- **Сдвиг макета (CLS):** В `StreamStatus.tsx` устранено прыгание шапки при загрузке статуса за счет введения серого состояния "Загрузки".
- **Скроллбар:** В `globals.css` добавлены исправления для горизонтального скроллбара и его угла пересечения.
